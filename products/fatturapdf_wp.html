Frequently Asked Questions

Does this work with any WordPress theme?

Yes. The plugin runs entirely in the WordPress admin dashboard and uses native WP APIs. Theme conflicts are extremely rare.

Can I generate Italian Electronic Invoices (Fattura Elettronica SDI)?

No. This plugin generates professional PDF documents (invoices, quotes, receipts). It is NOT compliant with the Italian SDI electronic invoicing system required by law. Use dedicated accounting software for SDI compliance.

How do I upload my company logo?

Use the "Upload Logo" button inside the invoice creation panel. It integrates with WordPress's native media library for secure, optimized handling.

Is technical support available?

Yes. For installation, setup, or usage questions, contact me via the support form. I respond within 24h with actionable guidance.

Is this suitable for non-Italian users?

Absolutely. While optimized for Italian tax formats, the core features (WYSIWYG editor, PDF generation, themes, logo upload) are universal. You can manually adapt default text fields to match your country's requirements.
